# Tools

* [just](https://github.com/casey/just) - better command runner than make
* [hoard](https://github.com/Hyde46/hoard) - command organizer and runner
* [gaze](https://github.com/wtetsu/gaze) - watch and run files on save
* [glances](https://github.com/nicolargo/glances) - python-based top replacement
* [chezmoi](https://www.chezmoi.io) - dotfile management
* [yadm](https://yadm.io/docs/overview) - dotfile management
* [Various MacOS configuration settings](https://mths.be/macos)
